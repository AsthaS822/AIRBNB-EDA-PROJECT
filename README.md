Retail Sales Analysis Dashboard
Overview

This project is a Retail Sales Analysis Dashboard created using Microsoft Power BI and Python.
I built this project to analyze retail sales data and understand business performance across different countries, cities, regions, and product categories.

The dashboard helps in finding important insights related to sales, profit, yearly growth, and regional performance using interactive visualizations.

Objectives
Analyze total sales and profit
Track yearly sales trends
Compare sales and profit across categories
Identify top-performing cities and countries
Analyze India sales contribution compared to global sales
Understand regional and state-wise performance
Create interactive and easy-to-understand dashboards
